# 🤖 Agency.ai

🚀 Built as part of my frontend portfolio to showcase real-world project skills.

Agency.ai is a modern AI-powered digital agency website designed to showcase services like advertising, content marketing, content writing, and social media management. The project focuses on delivering a visually appealing, responsive, and engaging user experience using smooth animations and modern UI design.

---

## ✨ Features

- 🎨 Modern and clean UI design
- 🌙 Dark mode support
- ⚡ Fully responsive (mobile + desktop)
- 🎬 Smooth animations using Framer Motion
- 🧩 Reusable React components
- 📊 Service-based layout for digital agency
- 🚀 Fast performance with Vite

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- Tailwind CSS
- Framer Motion
- HTML5 & CSS3
- Vite
